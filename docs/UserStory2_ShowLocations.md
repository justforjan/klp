As a website visitor, I want to be able to get more information about the locations of events, so that I can plan my visit accordingly.
I want to see details such as the address and events that take place at each location.

Acceptance Criteria:
1. There is a separate locations page accessible from the main navigation menu.
2. The locations page displays a list of event locations with their name and address.
3. Each location entry includes a calender like display of all events at that location.
5. The locations are sorted alphabetically by default.
6. User can click on the location link on an event listing to be taken to the corresponding location details.
