As a website visitor, I want to see a list of event occurences that can be filtered by date range and name/description, so that I can easily find events that interest me.

Acceptance Criteria:
1. The events page displays a list of events with their name, description, date, and location.
2. There is a date range filter that allows users to select a start date and an end date to view events occurring within that range.
3. There is a text search filter that allows users to search for events by name or description.
4. The list of events updates dynamically based on the selected date range and search text.
5. If no events match the filter criteria, a message is displayed indicating that no events were found.
6. The events are sorted by date in ascending order by default.
7. The user interface is responsive and works well on both desktop and mobile devices.
