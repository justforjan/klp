As a website visitor, I want to be able to mark exhibitions as favourites, so that I can easily access them later.

Acceptance Criteria:
1. There should be an option to mark exhibitions as favourites on the exhibition carousel on the location page.
2. When an exhibition is marked as a favourite, it should be added to the user's list of favourited exhibitions.
3. The user should be able to view their list of favourited exhibitions on the favorites page.
4. The exhibitions on the favourite page should be grouped by location
5. The favourite exhibitions should be part of the PDF export functionality, allowing users to export their favourited exhibitions as a PDF.
