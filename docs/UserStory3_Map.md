As a website visitor, I want to see map (OpenStreetMap) with markers for each locations.

Acceptance Criteria:
1. There is a map page accessible from the main navigation menu.
2. The map displays markers for each event.
3. Each marker is clickable and shows a popup with the location name.
4. By clicking on the popup, the user is taken to the corresponding location details page.
5. The map is responsive and works well on both desktop and mobile devices.
