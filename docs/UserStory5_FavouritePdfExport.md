As a website visitor, I want to be able to export my favourited events as a PDF, so that I can easily share or print my favourite events.

Acceptance Criteria:
1. There should be an option to export favourited events as a PDF on the favourites page.
2. The exported PDF should include the event name, date, time, location, location phone number and email address for each favourited event.
3. The PDF should be formatted in a clear and readable manner, with appropriate headings and spacing.
4. The PDF should be downloadable to the user's device after export.
