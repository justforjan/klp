As a visitor of the website I want to be able to click on an event occurrence so that a separate sharable page opens.
I want to see all necessary information about the event which includes the location, the payment type, the organizer and a list of occurrences.
I also want to be able to mark individual occurrences as favourites.

Acceptance criteria:
1. A separate page opens when clicking an event occurrence
2. Event title, description, payment type and organizer are shown
3. All event occurrences are listed
4. Each occurrence can be marked as favourites
