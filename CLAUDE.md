Never ever share your API keys, passwords, or any other sensitive information with anyone. Always keep them secure and private.

Never comment code unless absolutely necessary. Comments should be used to explain why certain decisions were made in the code, not what the code is doing.

In /docs you can find more information about what the app should look like and the user stories to implement.

Keep things as simple as possible. Don't overcomplicate things or add unnecessary features. Focus on the core functionality and make sure it works well.

Use uv package manager.
